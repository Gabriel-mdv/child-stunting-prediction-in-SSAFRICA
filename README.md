# CHW Child Stunting Prediction Tool

A web-based tool for Community Health Workers to predict child stunting risk
across Sub-Saharan Africa using machine learning.

---

## Setup

### 1. Install Python dependencies

```bash
pip install -r requirements.txt
```

### 2. Place your DHS data files

Make sure your folder structure matches exactly:
```
dataset/NGKR7BDT/NGKR7BFL.DTA
dataset/RWKR81DT/RWKR81FL.DTA
dataset/ETKR71DT/ETKR71FL.DTA
```

---

## Running the Tool (3 steps)

### Step 1 — Clean and merge the data
```bash
python 1_data_pipeline.py
```
This will create `data/combined_data.csv` (~29,000 rows)

### Step 2 — Train the model
```bash
python 2_model_training.py
```
This will train XGBoost, evaluate it, and save:
- `models/xgboost_model.pkl`
- `models/shap_explainer.pkl`
- `models/model_report.txt`

### Step 3 — Run the web tool
```bash
python 3_api.py
```
Then open your browser at: **http://localhost:8000**

---

## What the Tool Does

1. CHW selects country and enters 12 simple fields
2. XGBoost model predicts stunting probability
3. Tool returns:
   - Risk level (HIGH / MODERATE / LOW)
   - Probability score (e.g. 74%)
   - Top 3 risk factors (powered by SHAP)
   - Recommended action (refer / monitor / continue)
